## 🚀 Como contribuir com um prompt

1. Use o prompt em uma tarefa real da agência
2. Se funcionou bem → poste no canal **#ai-prompts** no Slack
3. Use o formato padrão:

```
ÁREA: [Atendimento / Criação / Social / Mídia / BI / Influenciadores / Ops / Projetos]
PROBLEMA: [Qual tarefa você estava fazendo]
PROMPT:
---
[Cole o prompt aqui]
---
RESULTADO: [Descreva o que funcionou bem]
```

4. O responsável da área sobe pro banco na próxima Lab Session (quinzenal)
